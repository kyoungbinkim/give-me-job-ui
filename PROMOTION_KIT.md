# give-me-job 홍보 키트

## 한 줄 소개

give-me-job은 이력서와 채용공고를 근거로 자기소개서 작성, 검수, 면접 준비를 돕는 한국 취업 AI 에이전트입니다.

## 짧은 게시글

자소서 쓸 때 제일 위험한 게 "그럴듯하지만 내 경험이 아닌 말"이라고 생각해서 give-me-job을 만들었습니다.

이력서와 채용공고를 분석해서 JD 키워드, 내 경험 근거, 자소서 초안, HR 관점 검수, 면접 꼬리 질문까지 한 번에 준비하는 취업 AI 에이전트입니다.

설치:

```bash
npx give-me-job install
```

소개 페이지: https://give-me-job.vibenest.net/
GitHub: https://github.com/kyoungbinkim/give-me-job

## 커뮤니티용 긴 게시글

취준하면서 자기소개서를 쓰다 보면 "잘 써야 한다"보다 더 어려운 게 "내가 실제로 한 일 안에서 설득력 있게 써야 한다"는 점이었습니다. 그래서 한국 취업 준비 흐름에 맞춘 AI 에이전트 `give-me-job`을 만들었습니다.

핵심은 허위 경험을 만들어내는 게 아니라, 이력서와 채용공고를 같이 보고 근거 있는 자기소개서와 면접 대비 자료를 만드는 것입니다.

주요 기능:

- 이력서 메모를 재사용 가능한 근거로 정리
- 채용공고의 요구 역량, 평가 기준, 키워드 분석
- 내 경험 안에서 자기소개서 초안 작성
- 과장, 근거 부족, 회사명 복붙 같은 제출 전 리스크 검수
- 자소서 주장별 예상 꼬리 질문과 답변 근거 준비

설치는 아래처럼 할 수 있습니다.

```bash
npx give-me-job install
```

소개 페이지: https://give-me-job.vibenest.net/
GitHub: https://github.com/kyoungbinkim/give-me-job

피드백, 이슈, PR 모두 환영합니다.

## Reddit / Hacker News 스타일

I built `give-me-job`, an open-source AI agent for Korean job applications.

It analyzes a resume and a job description, then helps generate grounded cover letter drafts, review risky claims, and prepare interview follow-up questions. The goal is not to invent fake experience, but to keep the output tied to the user's actual resume evidence.

Install:

```bash
npx give-me-job install
```

Landing page: https://give-me-job.vibenest.net/
